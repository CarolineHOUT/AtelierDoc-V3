# Correctif navigation retour

## Ajouté

- Barre de retour visible sur toutes les pages secondaires utilisant le Shell principal.
- Bouton `Retour` qui ramène à l’accueil sans perdre les données de l’atelier.
- Positionnement en flux desktop pour ne pas chevaucher les titres.
- Barre sticky mobile pour garder le retour disponible même avec le menu latéral masqué.

## Vérification

- `npm run build` exécuté avec succès.
